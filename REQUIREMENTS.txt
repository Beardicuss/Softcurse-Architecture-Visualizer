================================================================================
SOFTCURSE ARCHITECTURE VISUALIZER - SYSTEM REQUIREMENTS
================================================================================

This document outlines the minimum and recommended hardware and software 
specifications for running the Softcurse Architecture Visualizer desktop 
application for optimal performance, especially when analyzing large codebases 
with 10,000+ files and complex AI interactions.

--------------------------------------------------------------------------------
1. HARDWARE REQUIREMENTS
--------------------------------------------------------------------------------

[ MINIMUM SPECIFICATIONS ]
- Processor (CPU) : Intel Core i3 / AMD Ryzen 3 (Dual-core, 2.0 GHz+)
- Memory (RAM)    : 4 GB
- Storage         : 300 MB available space for installation
- Graphics (GPU)  : Integrated graphics supporting WebGL 2.0
- Display         : 1280 x 720 minimum resolution
- Architecture    : 64-bit (x64)

[ RECOMMENDED SPECIFICATIONS ]
- Processor (CPU) : Intel Core i7 / AMD Ryzen 7 (Quad-core, 3.0 GHz+)
- Memory (RAM)    : 8 GB+ (for parsing large 10K+ file repositories)
- Storage         : 1 GB+ Solid State Drive (SSD) for faster local indexing
- Graphics (GPU)  : Dedicated GPU (NVIDIA GTX 1060+ / AMD RX 580+)
- Display         : 1920 x 1080 (Full HD) or higher resolution
- Architecture    : 64-bit (x64)

* Note on Graphics: The application utilizes Sigma.js WebGL rendering capable 
  of drawing tens of thousands of graph nodes at 60fps. A GPU with WebGL 2.0 
  support is essential for smooth graph navigation and process flow tracing.

--------------------------------------------------------------------------------
2. SOFTWARE REQUIREMENTS
--------------------------------------------------------------------------------

[ OPERATING SYSTEM ]
- Supported OS: Windows 10 (Version 1803 or later) and Windows 11
- Architecture: 64-bit only

[ DEPENDENCIES ]
1. Microsoft Edge WebView2 Runtime
   - Description: Powers the embedded React/Vite user interface.
   - Status: Pre-installed on Windows 11. 
   - Note: If missing on older Windows 10 builds, the installer or prompt 
     will guide you to download it from Microsoft's official site.

2. .NET 8.0 Desktop Runtime (Embedded)
   - Status: The application is packaged as a "Self-Contained" executable. 
     You DO NOT need to install the .NET runtime manually on your machine.

--------------------------------------------------------------------------------
3. NETWORK CONFIGURATIONS
--------------------------------------------------------------------------------

The application is designed to be "Offline-First" for maximum privacy. Your 
local source code is NEVER uploaded. However, an internet connection is 
required for specific features:

1. Softcurse AI Chat Agent
   - Requires internet access to communicate with external Large Language Model 
     providers (e.g., Google Gemini, OpenAI, Anthropic, OpenRouter).
   - Exempt: If using 'Ollama' with local models, internet is not required.

2. GitHub Repository Cloning 
   - Requires internet access to download repositories via https://github.com/ URL.

3. External Component Connections
   - Connecting to a remote Softcurse Server (via Server Tab) requires LAN or 
     WAN access depending on the server's location.

--------------------------------------------------------------------------------
4. PERMISSIONS
--------------------------------------------------------------------------------

- Standard User Permissions are sufficient to run the application.
- File System Access: The application requires permission to read directories 
  you explicitly drop or browse in order to parse the source code.
- Installer: The Inno Setup installation package requires Administrator 
  (Elevated) privileges during the initial installation process.
